A D Flip-Flop (Data or Delay Flip-Flop) is a fundamental sequential logic circuit used to store a single bit of digital data. It features a data input (d), an asynchronous reset input (rst), a clock input (clk), and a data output (q). The core rule is that the output captures the state of the data input line only at the exact moment a rising clock edge occurs, holding that value steady until the next active clock trigger. If the reset line activates, the memory clear function takes immediate priority.

In this simulation, the reset line is kept inactive (rst=0) to let data pass normally. Internally, the code runs a process block that monitors changes on the clock and reset lines. The logic structure explicitly checks the reset condition first: if rst is high, the output drops to 0 instantly. If reset is quiet, the conditional statement checks for a rising clock edge using a clock event detector. When a valid transition happens, the internal memory latch executes a state transfer, updating the stored value directly from the data pin (q <= d).

Looking closely at the waveforms, the output track (q) mimics the data line transitions but only changes its state in perfect synchronization with the rising edges of the clock signal.
