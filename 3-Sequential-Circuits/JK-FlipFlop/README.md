A JK Flip-Flop is a versatile sequential logic circuit used as a standard memory cell to store a single bit of data. It features two control inputs (j and k), an asynchronous reset input (rst), a clock input (clk), and a data output (q). The core rule is that it modifies its output on a rising clock edge based on four distinct input combinations: holding the current state (00), resetting to zero (01), setting to one (10), or completely reversing/toggling the output bit (11).

In this simulation, the reset line is kept inactive (rst=0), while the control inputs are set to j=1 and k=0. This specific combination represents the "Set" command. Internally, the code runs a process block that triggers with any change on the input lines. When a rising clock edge event is detected, the logic passes through the conditional statements, skips the reset and hold branches, and executes the state transfer statement that forces the output high (q <= 1).

The waveform graph visually verifies this edge-triggered behavior over the timeline. The clock track (clk) shows a regular pulsing square wave signal. Looking at the waveforms up to the 300 ns mark, input j stays high while input k remains flat at zero. Consequently, the output track q responds to the clock edge by stepping upward into a steady high state (1). This clear graphical output confirms that the circuit accurately evaluates input commands and performs bit storage exactly as designed.
