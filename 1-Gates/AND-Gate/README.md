# AND Gate using VHDL

This repository contains the VHDL implementation of an **AND Gate**. The design was written in **VHDL** and functionally verified using **ModelSim**.

An **AND Gate** is a basic digital logic gate that produces a HIGH (`1`) output only when both inputs are HIGH. If either input is LOW, the output remains LOW.
