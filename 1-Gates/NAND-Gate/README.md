# NAND Gate using VHDL

This repository contains the VHDL implementation of a **NAND Gate**. The design was written in **VHDL** and functionally verified using **ModelSim**.

A **NAND Gate** is the inverse of an AND gate. It produces a LOW (`0`) output only when both inputs are HIGH. Otherwise, the output remains HIGH.
