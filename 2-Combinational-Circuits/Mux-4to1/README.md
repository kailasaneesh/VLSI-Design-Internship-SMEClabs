# 4:1 Multiplexer using VHDL

This repository contains the VHDL implementation of a **4:1 Multiplexer**. The design was written in **VHDL** and functionally verified using **ModelSim**.

A **4:1 Multiplexer** selects one of four input signals and forwards it to a single output. The selected input depends on the values of the two select lines.

## Truth Table

| S1 | S0 | Output |
|----|----|--------|
| 0 | 0 | I0 |
| 0 | 1 | I1 |
| 1 | 0 | I2 |
| 1 | 1 | I3 |

## Software Used

- VHDL
- ModelSim

## Key Concepts

- Designing combinational logic circuits
- Understanding data selection using multiplexers
- Simulating and verifying the design using ModelSim
