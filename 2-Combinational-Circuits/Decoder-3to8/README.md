A 3-to-8 line decoder is a combinational logic circuit that converts a 3-bit binary input into one of eight unique outputs. It accepts three input signals (X, Y, Z) to create 8 possible binary combinations. The fundamental rule of the circuit is that only the single output line corresponding to the decimal value of the input code becomes active-high (1), while the remaining seven output lines are forced low (0).

When the inputs are set to X=1, Y=1, Z=1, they form the binary code 111, which represents the decimal number 7. Internally, the circuit uses NOT gates to invert the inputs, making X1=0, Y1=0, Z1=0. Because the equations for outputs D0 through D6 rely on at least one of these inverted signals, their concurrent AND gates evaluate to 0. However, the logic equation for D7 reads (X AND Y AND Z), which calculates directly to 1 (1 AND 1 AND 1 = 1).

Consequently, the output lines D0 through D6 remain completely flat at the zero reference line, while D7 is the solitary output signal that steps upward into a high state. This distinct output behavior confirms that the decoder design routes signals with absolute accuracy.
