A Full Adder is a combinational logic circuit that performs the arithmetic addition of three single-bit binary inputs: two data bits (a, b) and a carry-in bit (cin) received from a previous stage. The core rule is that it mathematically adds these bits together to produce a 2-bit result split across two outputs: a Sum bit and a Carry-Out (cout) bit.In this simulation, all three inputs are driven high (a=1, b=1, cin=1), which mathematically equals $1 + 1 + 1 = 3$. In binary language, the decimal number 3 is written as 11. Internally, the Sum gate uses an XOR operation (1 XOR 1 XOR 1), which evaluates to 1. Simultaneously, the Carry-Out logic checks if any two or more inputs are active using AND/OR combinations, which also evaluates to 1 because all pair terms are satisfied.The waveform graph visually verifies this arithmetic calculation: all three input tracks (a, b, cin) sit high up on the timeline.
